# Linear Regression – California Housing

This repository contains a small project from my **Statistical Methods** course.

The project includes:

- An implementation of a **Linear Regression class written from scratch in Python**
- A short **report/notebook applying the model to the California Housing dataset**

The goal of the project is to demonstrate the fundamentals of linear regression by implementing the model manually and using it to analyze a real dataset.

The report performs basic data exploration, trains the custom regression model, and evaluates its predictive performance on housing prices.
